2.Set JAVA_HOME and Path by clicking environment variables

6.Install Google cloud plugin from Eclipse marketplace(under Help)

7.Install Google cloud SDK --> https://cloud.google.com/sdk/docs/#install_the_latest_cloud_sdk_version
